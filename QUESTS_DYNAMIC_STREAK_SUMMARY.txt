HOW IT WORKS
================================================================================

Data Flow:
  1. User visits /quests
  2. QuestsController fetches streak from database
  3. Streak data passed to Quests.vue
  4. StreakCard component renders with real data
  5. Next day user logs in
  6. Middleware updates streak in database
  7. User visits /quests again
  8. StreakCard shows updated values

Daily Update:
  Day 1: Current = 1, Longest = 1
  Day 2: Current = 2, Longest = 2
  Day 3 (skip)
  Day 4: Current = 1 (reset), Longest = 2 (preserved)
  Day 5: Current = 2, Longest = 2
  ...continues...

Same Day:
  First login:    Streak updates (middleware runs)
  Same day again: No change (prevents cheating)
